.. _Trace4.0-depth_attributes:

**depth** -- estimated depth
----------------------------

This attribute gives the depth of the event
origin. In stassoc this may be an analyst estimate.

* **Field width:** 9
* **Format:** %9.4f
* **Null:** -999.0000
* **Units:** Kilometers
* **Range:** depth >= 0.0 && depth < 1000.0
