**stime** -- origin time error
------------------------------

This attribute denotes the time uncertainty that
accompanies the location.  The level of confidence is
specified by conf.  See smajax, sminax, and sdepth.

* **Field width:** 8
* **Format:** %8.2f
* **Null:** -1.00
* **Units:** Seconds
* **Range:** stime >= 0.0
