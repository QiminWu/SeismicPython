**drdepth** -- derived event depth
----------------------------------

Derived event depth; Corresponds to the
derived-depth parameter in the USGS/NEIC GeoJSON
data feed for moment tensors.

* **Field width:** 9
* **Format:** %9.4f
* **Null:** -999.0000
* **Units:** Kilometers
* **Range:** depth >= 0.0 && depth < 1000.0
