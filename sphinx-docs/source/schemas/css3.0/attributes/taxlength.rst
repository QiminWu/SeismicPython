**taxlength** -- moment tensor t-axis component
-----------------------------------------------

Moment Tensor Component; Corresponds to the
t-axis-length parameter in the USGS/NEIC GeoJSON
data feed for moment tensors.

* **Field width:** 12
* **Format:** %12.5e
* **Null:** -9.99999e+99
