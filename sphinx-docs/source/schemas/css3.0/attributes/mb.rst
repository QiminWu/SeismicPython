**mb** -- body wave magnitude
-----------------------------

This is the body wave magnitude of an event.  Associated
with this attribute is the identifier mbid which points to
magid in the netmag relation.  The information in that
record summarizes the method of analysis and data used.
See imb, iml, ims, magnitude, magtype, ml and ms.

* **Field width:** 7
* **Format:** %7.2f
* **Null:** -999.00
