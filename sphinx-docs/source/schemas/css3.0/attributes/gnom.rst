**gnom** -- nominal scaling factor
----------------------------------

This is the nominal scaling factor of this stage.  It can
have units associated with it if the stage is a sensor or
have no units if the stage is a filter or amplifier.

* **Field width:** 10
* **Format:** %10.5g
* **Null:** 0
* **Range:** gnom > 0.0
