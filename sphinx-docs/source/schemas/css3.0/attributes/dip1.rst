**dip1** -- Dip of first nodal plane
------------------------------------

Dip of First Nodal Plane; In degrees (for sign conventions
see Aki and Richards, 1980, p.  106).  In fplane, as
determied.  In moment, refers to best-fitting double
couple (see bestdc).

* **Field width:** 5
* **Format:** %5.1f
* **Null:** 999.9
