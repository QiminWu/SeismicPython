**azimuth** -- observed azimuth
-------------------------------

This is the estimated station-to-event azimuth measured
clockwise from north.  Azimuth is estimated from f-k or
polarization analysis.  In stassoc, the value may be an
analyst estimate.

* **Field width:** 7
* **Format:** %7.2f
* **Null:** -1.00
* **Units:** Degrees
* **Range:** azimuth >= 0.0 && azimuth < 360.0
