**seaz** -- station to event azimuth
------------------------------------

This attribute is calculated from the station and event
locations.  It is measured clockwise from North.

* **Field width:** 7
* **Format:** %7.2f
* **Null:** -999.00
* **Units:** Degrees
* **Range:** seaz >= 0.0 && seaz < 360.0
