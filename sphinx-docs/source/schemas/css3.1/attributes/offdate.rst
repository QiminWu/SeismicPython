.. _css3.1-offdate_attributes:

**offdate** -- Julian off date
------------------------------

This attribute is the Julian Date on which the station or
sensor indicated was turned off, dismantled, or moved.
See ondate.

* **Field width:** 18
* **Format:** %18ld
* **Null:** -1
* **Range:** offdate >= 1900001 && offdate <= 2100000
