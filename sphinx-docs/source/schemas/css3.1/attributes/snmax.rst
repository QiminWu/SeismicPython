**snmax** -- maximum north-south slowness (s/km)
------------------------------------------------

This is a maximum north-south slowness range.

* **Field width:** 7
* **Format:** %7.4f
* **Null:** -9.9999
* **Units:** Seconds/km
* **Range:** snmax > -9.9999 && snmax <= 9.9999
