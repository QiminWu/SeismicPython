**filter** -- filter description
--------------------------------

This is a character string filter specification.  (e.g.
"BW 1.0 5 5.0 5" for a Butterworth bandpass filter from 1
to 5 hz with 5 poles at each end)

* **Field width:** 30
* **Format:** %-30s
* **Null:** -
