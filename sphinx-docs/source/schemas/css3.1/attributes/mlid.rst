**mlid** -- ml magid
--------------------

This stores the magid for a record in netmag.  Mlid is a
foreign key joining origin to netmag where origin.  mlid =
netmag.  magid.  See magid, sid and mbid.

* **Field width:** 12
* **Format:** %12ld
* **Null:** -1
* **Range:** mlid > 0
