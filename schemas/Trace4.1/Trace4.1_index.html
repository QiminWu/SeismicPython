Trace4.1 – Trace Manipulation and Processing Database¶

Detail¶

This database provides the parameter

information needed during standard processing of seismic waveform data.

It is largely derived from the CSS 3.1 database, with additions which address specific trace manipulation issues, and the representation of traces in memory. The information contained in the central trace relation represents a join of many tables, in particular wfdisc, site, sitechan, sensor and instrument.

This is an update to the Trace4.0 schema to bew compatible with the various larger strings in css3.1
