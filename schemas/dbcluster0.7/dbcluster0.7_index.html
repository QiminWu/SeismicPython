dbcluster0.7 – Centralized registry of databases¶

Attributes¶

C | D | E | L | N | S | T | V | all

Relations¶

C | all

Detail¶

This schema provides a centralized registry of all
databases available from a given filesystem (possibly a composite filesystem mounted from several different sources and all visible from the same root location). The schema supports clusters of databases with regular naming conventions and divisions based (usually) on time intervals.
