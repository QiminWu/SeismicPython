description – short description of database or cluster¶

This field gives a short description of a database or database cluster, e.g. “master site database”, “backup site database”, “PDE Catalog”, “main real-time auto processing database”, etc., as appropriate for the operational environment.

  • Field width: 80
  • Format: %-80s
  • Null: -
