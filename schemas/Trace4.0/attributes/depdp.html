depdp – estimated depth from depth phases¶

This is a measure of event depth estimated from a depth phase or an average of several depth phases. Depth is measured positive in a downwards direction starting from the earth’s surface. See ndp.

  • Field width: 9
  • Format: %9.4f
  • Null: -999.0000
  • Units: Kilometers
  • Range: depdp >= 0.0 && depdp < 1000.0
