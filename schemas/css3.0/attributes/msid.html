msid – ms magid¶

This stores the magid for a record in netmag. Msid is a foreign key joining origin to netmag where origin. msid = netmag. magid. See magid, mlid and mbid.

  • Field width: 8
  • Format: %8ld
  • Null: -1
  • Range: msid > 0
