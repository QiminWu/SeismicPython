ounits – output units of stage (e.g. V, counts, ...)¶

This is the output units which are appropriate for the stage. For a seismometer the output units will be V, for a analog-to-digital converter the output units will be counts.

  • Field width: 16
  • Format: %-16s
  • Null: -
