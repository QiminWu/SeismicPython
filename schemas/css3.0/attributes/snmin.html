snmin – minimum north-south slowness (s/km)¶

This is a minimum north-south slowness range.

  • Field width: 7
  • Format: %7.4f
  • Null: -9.9999
  • Units: Seconds/km
  • Range: snmin > -9.9999 && snmin <= 9.9999
