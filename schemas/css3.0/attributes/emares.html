emares – incidence angle residual¶

This attribute is the difference between an observed emergence angle and the theoretical prediction for the same phase, assuming an event location as specified by the accompanying orid.

  • Field width: 7
  • Format: %7.1f
  • Null: -999.0
  • Units: Degrees
  • Range: emares >= -90.0 && emares <= 90.0
