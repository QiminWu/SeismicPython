hang – horizontal angle¶

This attribute specifies the orientation of the seismometer in the horizontal plane, measured clockwise from North. For a North-South orientation with the seismometer pointing toward the north, hang=0.; for East-West orientation with the seismometer pointing toward the west, hang=270. See vang.

  • Field width: 6
  • Format: %6.1f
  • Null: -999.9
  • Units: Degrees
  • Range: hang >= 0.0 && hang <= 360.0
