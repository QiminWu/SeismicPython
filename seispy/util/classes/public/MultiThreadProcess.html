seispy.util.MultiThreadProcess – MultiThreadProcess class¶

class seispy.util.MultiThreadProcess(inputter, main_processor, outputter, extra_args=None, config_params=None)[source]¶

A class to facilitate multi-threaded processing.

Public Methods Private Methods
start()  
Parameters:
  • inputter (generator) – a generator that will yield objects to process
  • main_processor (function) – a function to process objects on the input queue and put process result objects on the output queue
  • outputter (function) – a function to output the process result objects
  • extra_args (dict) – a dict to provide initialization arguments and keyword arguments to inputter, main_processor and outputter
  • config_params (dict) – a dict to provide configuration parameters
Raises:
  • TypeError – if arguments are of invalid type
  • ValueError – if arguments are of invalid type
extra_args
Keyword Description
input_init_args a list of arguments to be passed straight to inputter generator.
input_init_kwargs a dict of keyword arguments to be passed straight to inputtter generator.
main_init_args a list of positional arguments to be passed straight to main_processor function, immediately following object to be processed.
main_init_kwargs a dict of keyword arguments to be passed straight to main_processor function.
output_init_args a list of positional arguments to be passed straight to outputter function, immediately following processed object.
output_init_kwargs A dict of keyword arguments to be passed straight to outputter function.
start()[source]¶

Start multi-threaded processing. This method will block until processing is complete and all threads have been terminated.

New in version 0.0alpha.
