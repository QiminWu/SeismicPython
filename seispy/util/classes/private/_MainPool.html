is_alive()[source]¶

Return True if main processing pool is still alive, else return False.

signal_kill()[source]¶

Send kill signal to main processing pool. Main processing pool will not terminate until input queue is empty.

start()[source]¶

Start input process.

_target_wrapper(thread_id)[source]¶

A wrapper to take objects off the input queue, pass them to the main processing function and place the resulting processed object on the output queue.

New in version 0.0alpha.
