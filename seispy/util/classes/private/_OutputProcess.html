class seispy.util._OutputProcess(target, output_q, *args, **kwargs)[source]¶

A wrapper class for the output thread.

Parameters:
  • target (function) – target function to output processed objects
  • input_q (Queue.Queue) – a Queue.Queue object containing processed objects
  • *args (tuple) – a dereferenced tuple of positional arguments to be passed straight through to target output function
  • **kwargs (dict) – a double-dereferenced dict of keyword arguments to be passed straight through to target output function
is_alive()[source]¶

Return True if output process is still alive, else return False.

signal_kill()[source]¶

Send kill signal to output process. Output process will terminate only when the output queue is empty.

start()[source]¶

Start output process.

_target_wrapper()[source]¶

A wrapper to take processed objects off the output queue and pass them to the output function.
