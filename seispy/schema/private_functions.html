seispy.schema Private Functions¶

seispy.schema._draw_table(iterator, header=None, schema='NULLSCHEMA', ncol=6)¶

Return a Sphinx-style complex table drawn around schema elements.

Parameters:
  • iterator (list) – list or tuple of _Attribute or _Relation objects
  • header (str) – optional header string to place at top of table
  • schema (str) – schema to which attributes/relations belong
  • ncol (int) – number of columns in table
seispy.schema._sort_key(obj)¶

Return value of object sort key.

Parameters:obj (object) – a instance of a class derived from _SchemaElement or a tuple of such classes.
Returns:value of obj‘s (or first instance in a tuple‘s) name attribute
Return type:str

This method is a simple wrapper to return the name attribute of the obj argument. If obj is a tuple, return the name attribute of the first element.
