seispy.schema._SchemaElement – _SchemaElement class¶

class seispy.schema._SchemaElement(**kwargs)[source]¶

Bases: object

Base class to define inheritable behaviour of schema elements (attributes/relations).

This class defines fuctionality that is mutual to both _Attribute and _Relation classes. This behaviour consists of mapping key-value pairs to object attributes upon instantiation and the __str__ method.

Parameters:**kwargs (dict) – a double-dereferenced dict consisting of schema element keys and corresponding values parsed from schema definition
