**rect** -- rectilinearity
--------------------------

This attribute is a measure of signal
rectilinearity. The value is obtained from polarization
analysis of 3-component data.

* **Field width:** 7
* **Format:** %7.3f
* **Null:** -1.000
* **Range:** rect >= 0.0 && rect <= 1.0
