.. _css3.1-nettype_attributes:

**nettype** -- network type, array, local, world-wide, etc.
-----------------------------------------------------------

This 4 character string specifies what type of network (ar
= array), (lo = local area), (ww = world-wide) for the
given value of net.

* **Field width:** 4
* **Format:** %-4s
* **Null:** -
