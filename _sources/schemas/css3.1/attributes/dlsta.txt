**dlsta** -- datalogger station id
----------------------------------

A datalogger may insert some network/station id into its
output datastream; this field records that id.  For some
dataloggers, this id is a combination of a network code
and a station code, separated by an underscore.

* **Field width:** 16
* **Format:** %s
* **Null:** -
