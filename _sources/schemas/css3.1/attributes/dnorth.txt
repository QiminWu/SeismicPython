**dnorth** -- offset from array reference (km)
----------------------------------------------

This attribute gives the 'northing' or relative position
of array element north of the array center specified by
the value of refsta.  See deast.

* **Field width:** 10
* **Format:** %10.5f
* **Null:** 0.0000
* **Units:** Kilometers
* **Range:** dnorth >= -20000.0 && dnorth <= 20000.0
