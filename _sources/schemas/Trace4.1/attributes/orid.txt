**orid** -- origin id
---------------------

Each origin is assigned a unique
positive integer which identifies it in a data base. The
orid is used to identify one of the many hypotheses of the
actual location of the event.

* **Field width:** 12
* **Format:** %12ld
* **Null:** 0
* **Range:** orid > 0
