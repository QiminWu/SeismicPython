.. _Trace4.1-keyname_attributes:

**keyname** -- id name (arid, orid, etc.)
-----------------------------------------

This attribute contains the actual name
of a key whose last assigned numeric value is saved in
keyvalue.

* **Field width:** 15
* **Format:** %-15s
* **Null:** NONULL
* **Range:** keyname =~ /arid|chanid|commid|evid|inid/
