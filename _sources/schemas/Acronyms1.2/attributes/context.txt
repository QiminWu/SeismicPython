.. _Acronyms1.2-context_attributes:

**context** -- Acronym context
------------------------------

The context field contains an optional indication of the
context of an acronym. An example might be "NSF" if
the acronym is a National Science Foundation subproject.
The intent is to keep some subsettable consistency in
the context fields, where used, to help group acronyms
where possible.

* **Field width:** 80
* **Format:** %-80s
* **Null:** -
