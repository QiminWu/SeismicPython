**stream** -- stream number
---------------------------

The data stream identification number.  Many dataloggers
with internal digital signal processors can generate
multiple data streams with different filtering and and/or
sample rates using the same sensor as an input.  This
number is used in the calibration relation.

* **Field width:** 8
* **Format:** %8ld
* **Null:** -1
