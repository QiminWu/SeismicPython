.. _css3.0-drlat_attributes:

**drlat** -- derived event latitude
-----------------------------------

Derived event latitude; Corresponds to the
derived-latitude parameter in the USGS/NEIC GeoJSON
data feed for moment tensors.

* **Field width:** 9
* **Format:** %9.4f
* **Null:** -999.0000
* **Units:** Degrees
* **Range:** lat >= -90.0 && lat <= 90.0
